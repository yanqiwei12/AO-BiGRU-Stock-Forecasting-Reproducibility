# AO-BiGRU-Stock-Forecasting-Reproducibility
Reproducibility package for one-step-ahead stock price forecasting using an AO-optimized BiGRU model, including raw data, preprocessing, optimization, prediction, and backtesting code.
# AO-BiGRU Reproducibility Package

This repository contains the full reproducibility materials for the paper:
“Mitigating Investment Risk and Analyzing Future Stock Behavior: A Novel Approach in Financial Markets”.

## Objective
To reproduce the AO-BiGRU forecasting and backtesting results reported in the manuscript.

## Forecast target
The model predicts Close(t+1) using the observed feature vector at time t.

## Repository contents
- raw data
- processed modeling matrices
- preprocessing scripts
- AO/GA/ALO optimization code
- prediction outputs
- out-of-sample backtest files
- portfolio metrics and trade logs

## Reproducibility steps
1. Create the Python environment
2. Run data preparation
3. Run optimizer
4. Train final AO-BiGRU
5. Generate predictions
6. Run backtest
7. Compare outputs with the manuscript tables
